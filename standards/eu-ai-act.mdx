Complior maps all 108 deployer obligations from the EU AI Act (Regulation 2024/1689) to concrete features. Each obligation is tracked through the 7-step compliance pipeline.

<Info>
  **Enforcement date:** August 2, 2026. Fines: up to €35M or 7% of global turnover (Art.99).
</Info>

## Risk classes

| Class | Description | Complior determines via |
|-------|------------|----------------------|
| **Prohibited** | Banned practices (Art.5) | SDK 138-pattern matcher |
| **High** | Annex III systems (HR, healthcare, finance, education, law enforcement) | Autonomy level × project domain |
| **Limited** | Transparency obligations (Art.50) | Default for L3–L4 autonomy |
| **Minimal** | No specific requirements | Default for L1–L2 autonomy |
