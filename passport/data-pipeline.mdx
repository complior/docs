## Stage 1: What gets auto-filled

After `complior agent init`, these fields are populated from AST analysis:

- **Identity:** agent_id, name, display_name, description, version, timestamps
- **Autonomy:** type, level L1–L5, human_approval_gates count, unsupervised_actions count, killSwitchPresent
- **Tech Stack:** framework, model provider/id/deployment, data_residency
- **Permissions:** tools list, data_access (read/write/delete), denied actions
- **Compliance:** risk_class, applicable_articles, obligations met/pending
- **Logging:** actions_logged, retention_days
- **Lifecycle:** status ("draft"), next_review (created + 90 days)
- **Oversight:** (for high-risk/L3+ only) responsible_person, override_mechanism, escalation
- **Signature:** ed25519 auto-sign
